GoLab Molecule Naming Lab

Introduction

Using the GoLab Molecule Naming Lab you select from a list of numbered molecules, are shown the selected molecule diagram, and are asked to name it.

On the next page use the buttons to create the name of the molecule. Once you think you have correctly named the molecule check to see if you are correct.

There are 19 different molecules for you to name. Can you correctly name them all?
